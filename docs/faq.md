# FAQ

## Does this extension modify remote servers?
No. All changes happen locally in your browser.

## Will this affect other users?
No. The proxy rules only apply to your own Chrome environment.

## Can I enable multiple rules at the same time?
Yes. Rules are applied based on priority order.

## Does it support HTTPS?
Yes, as long as the local service is accessible.

## Is it similar to Charles or Fiddler?
Conceptually yes, but this extension is:
- Browser-only
- Fully visual
- Lightweight
- Focused on frontend workflows
